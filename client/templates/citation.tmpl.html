This WebApp is submitted as part of the project for CS 5200 Introduction to Database Management.

The Tech stack used for the Project is:
  1. DATABASE: MySQL 5.7.15
  2. NodeJS (Version: 4.5.0) as Server
  3. ExpressJS as Framework for building routes
  4. HTML5 + CSS3 + JS for frontend
  5. Boostrap for building a rich UI interface
  6. AngularJS for seameless routing and state persistence in the frontend
  7. SequelizeJS as the ORM for connecting and communicating with database.
